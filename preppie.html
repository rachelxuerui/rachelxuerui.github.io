Preppie
Designing and implementing AI powered interview preparation
This project was completed for the grad-level course: Advanced Web Design Studio, taught by Lydia Chilton at Columbia University. Our semester-long project was to solve a problem using generative AI. I decided to focus on solving the issue of a lack of conversational interview preparation resources.

PROBLEM
It is difficult for young professionals to prepare for behavioral interviews. They find that they are able to practice the most during the actual interview itself.
SOLUTION
Preppie takes in relevant inputs: your work experience, the role you are applying for, the company, and anything else you want to add to personalize your interview.

Preppie then simulates a live, remote interview experience, with a realistic AI interviewer.

Preppie will use generative AI to provide feedback on your responses to interviewer questions, including what went well, what to improve, and example responses.
CODE
I've coded this entire project using Flask (Python), HTML, CSS, and JavaScript.

If you're curious about my code, please reach out to me at rxm2101@columbia.edu for access to the GitHub repository. You need a Synthesia and an OpenAI API key for the code to run locally.
