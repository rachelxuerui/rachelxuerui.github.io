The Bulletin - Landing page
How might we excite and inform new users about The Bulletin?
I was tasked with redesigning the landing page for The Bulletin.

This project was short yet sweet - it only took me a few hours, but I got to create something simple and beautiful that I hope will excite new users to sign up for The Bulletin.
