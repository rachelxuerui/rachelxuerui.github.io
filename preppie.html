Designing and implementing AI powered interview preparation
This project was completed for the grad-level course: Advanced Web Design Studio, taught by Lydia Chilton at Columbia University.

Our semester-long project was to solve a problem using generative AI.

I decided to focus on solving the issue of a lack of conversational interview preparation resources.

Problem

It is difficult for young professionals to prepare for behavioral interviews. They find that they are able to practice the most during the actual interview itself.
Approach
Use the OpenAI API to generate interview questions based on a user's work experience, the role they are applying for, and any additional inputs
Use the Synthesia API to generate a realistic interviewer interface
Throughout the interview, use the Web Speech API to convert user response into text
At the end of the interview, use the OpenAI API to provide feedback on user responses from the interview
Solution
Preppie takes in relevant inputs: your work experience, the role you are applying for, the company, and anything else you want to add to personalize your interview.

Preppie then simulates a live, remote interview experience, with a realistic AI interviewer.

Preppie will use generative AI to provide feedback on your responses to interviewer questions, including what went well, what to improve, and example responses.

I've coded this entire project using Flask (Python), HTML, CSS, and JavaScript.
