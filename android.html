Android - redesigned
Making go-to-cart a primary action, and improving aspects of the Android navigation experience
Project overview
This was my secondary project for my internship with ExpertVoice.

The project started out simple, with the task of adding a cart icon to the Android experience. However, as I continued to design, I found more and more design flaws that needed to be fixed.

The end result is a new Android design that aligns more closely with the web and Apple experience, and is overall more intuitive and easy to use.

Problem
Android users can only access their carts by opening a kabob menu first, an unintuitive experience that deprioritizes the checkout process.
Original Android go-to-cart flow
Initial designs
I decided to scrap the simplest solution, adding a cart button to the top right, due to the overwhelming nature of the “icon salad”.

I decided to explore what removing the kabob menu would look like instead, placing its options into a menu appearing before the user profile.

This got good feedback. However, I still was not pleased with the overwhelming nature of all of the icons. I thus explored what it might look like if the navigation bars were split, a design more consistent with our Apple experience.
Too many icons in the top. Overwhelming. Icon salad.
Still visually overwhelming. Not enough physical space between icons.
Navigation bars are too bright. Distracts the user from the main content.
After presenting these designs to other engineers and designers, we decided the red was too vibrant of a color, and ultimately distracts users from the content we want them to focus on.
Final designs
The Android redesign is now live! Special thanks to the engineering team at ExpertVoice for implementing and testing my designs.
Cart icon is a primary action, accessible from all pages, encouraging users to checkout.
A page before a user’s profile listing options previously held in the kabob menu
A cleaner white design using red as a secondary color that is easier on the eyes and allows users to focus on shopping.
Next steps
A potential future design is an expansion of the search bar, encouraging more users to shop through an increased width of the search bar.
